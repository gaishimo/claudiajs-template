# Claudia.js Template

A boilerplate to develop Lambda (or Lambda & API Gateway) with Claudia.js.

## Stack

- Claudia.js
- Babel (ES2015 ES2016 ES2017)
- ESLint
- flow-type
- Mocha
- power-assert


## How to use

### Install

```
yarn install
```

### Test

```
yarn test
```

### Create Lambda function & API Gateway

```
yarn run create
```

### Update Lambda function & API Gateway

```
yarn run update
```
